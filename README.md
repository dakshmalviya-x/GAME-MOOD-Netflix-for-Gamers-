## ✨ About

GameMood is a full-stack PC game recommendation platform that helps users discover games according to their mood and preferences.

Users can:

* 🎯 Get mood-based game recommendations
* 🔥 Browse trending PC games
* ❤️ Save games to their wishlist
* 🔐 Register and login securely
* 💎 Upgrade to Premium
* 💰 Find the cheapest game deals using CheapShark API
* 🎨 Unlock beautiful neon themes

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/GameMood.git
```

## Client

```bash
cd client
npm install
npm run dev
```

## Server

```bash
cd server
npm install
npm start
```
